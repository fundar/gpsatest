== Description ==

POWr Form Builder allows you to create beautiful, mobile-responsive forms that you can edit **right in the page**. Add the widget to your site, or use the shortcode **[powr-form-builder]** to add forms to any post. 

* Surveys
* Polls
* Contact Forms
* Mailing List Signups
* Event Registrations
* Sales Leads
* Feedback Forms
* ...much more!

Enter an unlimited number of inputs in the form of text inputs, checkboxes, radio buttons, or select menus. Fully customize all language, fonts, and colors, and receive instant updates when a form is submitted.

The basic version of the POWr Form Builder is free! 

POWr plugins can be used an virtually any website, learn more at www.powr.io


Features:

* Unlimited number of form inputs, including text, text-areas, select fields, radio buttons, and checkboxes
* Immediate email notifications when a form is submitted
* Custom "thank you" message when form is submitted
* Option to require users to fill in fields
* Language support - customize text to use any language
* Fully customized fonts, colors, borders and background to fit your brand


== Installation ==

To install Form Builder:

1. Upload `powr-form-builder.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

To add Form Builder as a widget, go to **Appearance->Widgets**, and drag POWr Form Builder to where you'd like it to appear.

To add Form Builder within a post or a page, simply write the shortcode **[powr-form-builder]** (including the brackets) where you'd like your Form Builder to appear. Then visit your page!

Want a more detailed tutorial? Visit [How to add Form Builder to your Wordpress site](http://powr.io/tutorials/how-to-add-form-builder-plugin-to-your-wordpress-site)


== Frequently Asked Questions ==

= How can I add Form Builder to a blog post or page? =

Just add the shortcode **[powr-form-builder]** (including the brackets) in your post where you'd like your Form Builder to appear. Then visit your page!


= Why do you require me to sign up? =

Signing up is what allows us to track which widgets are yours, and allows you to edit them right in your page.


= Will my users see the settings icon in the corner of the Form Builder? =

No! You only see the settings icon because you own the widget. To temporarily hide it, click on your page and then type "p + down arrow" (the letter p followed by the down arrow). You can show the icon again by typing "p + up-arrow".
