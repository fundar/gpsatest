== Description ==

This plugin adds Google's reCAPTCHA form to your BuddyPress' registration page to keep your community spam-free! You can use out simple option panel to add reCAPTCHA credentials right from your Dashboard without editing any files.

This plugin is a forked version of <a target="_blank" href="http://wordpress.org/plugins/buddypress-recaptcha/">BuddyPress reCAPTCHA</a> plugin, which requires you to edit plugin core files to configure reCAPTCHA.

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure reCAPTCHA settigns from Settings > BuddyPress Captcha menu.
